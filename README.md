## PHP_Find_Array_Length

A simple PHP example demonstrating how to find the length of an array using the built-in count() function.

## Description

This small PHP script shows the easiest and most commonly used way to determine how many elements are present in an array. 
It is useful for beginners learning PHP arrays and for quick checks in real projects.

## Files

## Code Example

<?php

$array = ['One', 'Two', 'Three', 'Four', 'Five'];

$length = count($array);
var_dump($length);

?>

## Output

int(5)

## How It Works

The $array variable stores multiple values.

The count() function returns the total number of elements in the array.

var_dump() is used to display the data type and value of the result.

Usage

Run the script using a browser or the PHP CLI:

php index.php
